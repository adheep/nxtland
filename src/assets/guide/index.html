Introduction of Expleo Style guide

A style guide, when well followed, will
  1. Set the standard for UI quality across the platform;
  2. Promote consistency across website/mobile UI
  3. Give users a feeling of familiarity across the website/mobile
  4. Increase conversions owing to great UI/UX
Style Guides should be learned, understood, and implemented at all times on a project which is governed by one, and any deviation must be justified. A style guide (or manual of style) is a set of standards for the writing and design of documents, either for general use or for a specific publication, organization, or field.

Image Optimization

Optimization-Level1

  1. Consider jpg format. Suppose if need transparency, then only consider png(Only png8).
  2. Use Web Mode whenever saving jpg/png.
  3. Change quality dropdown value(Medium,low,High) for image quality, Whenever saving. Prefer Medium quality.

Optimization-Level2

  1. Use below any url for next level optimization
    • https://kraken.io/web-interface
    • https://tinypng.com/

GIF Optimization

  1. Consider less number of layers.
  2. Avoid gradient colors.
  3. To avoid patches, Matte color should apply while saving
